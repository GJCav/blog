## 格式覆盖

### 全宽图片

使用 `.img-w100` 类型，使用例：

``` markdown
![image-20240203120551485](imgs/image-20240203120551485.png){: .img-w100}
```

效果：![image-20240203120445902](imgs/image-20240203120445902.png){: .img-w100}



原理：

``` css
.img-w100 {
	max-width: 100% !important;
	max-height: none !important;
}
```
